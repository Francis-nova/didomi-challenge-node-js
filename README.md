# didomi-challenge-node-js


## Installation
Clone the repo to your computer 

Use the package manager of your choice to install the node_modules for this project.

```bash
npm install
```
or

```bash
npm install
```

## ESLint
```bash
npm run lint
```

## Test
```bash
npm run jest
```
